# Arkaius Digital Landing Page

Landing page premium V1 para Arkaius Digital, enfocada en automatización, inteligencia artificial, bots para WhatsApp, desarrollo web, soporte TI, redes e implementación de soluciones digitales.

## Ejecutar localmente

Abrir `index.html` directamente en el navegador o servir el directorio con un servidor estático:

```bash
python3 -m http.server 4173
```
